# Fahrduell Sounddateien

Lege hier die finalen MP3-Dateien fuer die Show-Sounds ab.

Vorbereitete Dateinamen:

- `quiz-start.mp3`
- `question-start.mp3`
- `countdown-warning.mp3`
- `answer-submitted.mp3`
- `answer-correct.mp3`
- `answer-wrong.mp3`
- `leaderboard.mp3`
- `winner.mp3`

Fehlende Dateien sind erlaubt. Die App faengt Abspielfehler bewusst ab.
